# Loan Approval: Preparing the Data for Decision Making

This project focuses on preparing and cleaning loan approval data to support accurate and reliable business decision-making.

## Project Overview

The objective of this project is to transform raw loan application data into a clean and structured dataset suitable for further analysis and predictive modeling.

## Tools & Technologies

- R Programming
- Google Colab (R Runtime)
- dplyr
- ggplot2

## Key Tasks Performed

- Data loading and inspection
- Handling missing values
- Data quality assessment
- Merging multiple datasets
- Data cleaning and preprocessing
- Exploratory data analysis
- Data preparation for decision-making

## Workflow

1. Import datasets
2. Explore dataset structure
3. Identify and handle missing values
4. Combine datasets
5. Clean and transform data
6. Validate data quality
7. Prepare final dataset for analysis

## Project Outcome

A cleaned and structured loan approval dataset ready for statistical analysis, reporting, and machine learning applications.

## Skills Demonstrated

- Data Cleaning
- Data Wrangling
- Exploratory Data Analysis (EDA)
- Data Quality Assessment
- R Programming
- Business Data Preparation
